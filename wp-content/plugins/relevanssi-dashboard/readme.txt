== Description ==


Dashboard Widgets for Relevanssi Search data. Of course requires the wonderful Relevanssi plugin too.


== Installation ==

1. Install the plugin from within the Dashboard or upload the directory `relevanssi-dashboard-widget` and all its contents to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
